# GitHub Issues

## Labels

### Priority Labels

| Label | Description |
|-------|-------------|
| `P1`  | High priority - should be addressed soon |
| `P2`  | Medium priority - address when time permits |
| `P3`  | Low priority - nice to have |

### Difficulty Labels

| Label | Description |
|-------|-------------|
| `D1`  | Easy - quick fix, minimal complexity |
| `D2`  | Medium - moderate effort required |
| `D3`  | Hard - significant effort, complex implementation |
